# Web Development Internship - Task 2 🚀

Welcome to my second web development project, created during my internship at **ApexPlanet Software Pvt. Ltd.** ## 📝 About the Project

This project showcases a modern, responsive web page featuring a **Glassmorphism Dark Theme**. It focuses on improving user interface design and adding interactive functionality using JavaScript, including client-side form validation and dynamic DOM manipulation.

## ✨ Key Features

* **Modern UI:** A beautiful Glassmorphism dark theme with blur effects, semi-transparent backgrounds, and radial gradients.
* **Responsive Design:** Utilizes CSS Grid to ensure the layout adapts seamlessly to different screen sizes.
* **Form Validation:** The "Send a Message" form uses JavaScript Regular Expressions (Regex) to validate email formats before submission.
* **Dynamic To-Do List:** A functional task tracker where users can type a task, add it to a list, and delete individual tasks dynamically.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Custom variables (CSS variables), Grid layout, hover animations, and backdrop-filter for the glass effect.
* **Vanilla JavaScript:** Event listeners, Regex for email validation, and DOM manipulation (creating and removing elements).

## 🚀 How to Run

1.  Clone or download this repository to your local machine.
2.  Double-click the `task2.html` file to open it in any modern web browser.
3.  Test out the email validation by entering incorrect and correct email formats.
4.  Try adding and deleting tasks in the Internship Task section!
